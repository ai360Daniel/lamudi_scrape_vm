# Lamudi Scraper (VM version)

Este repositorio contiene los archivos necesarios para ejecutar el scraping de Lamudi en máquinas virtuales de GCP.

## Requisitos
- Python 3.10+
- Google Cloud Storage Bucket access
- Service Account Key (.json)

## Archivos
- `scraper_functions.py`: Funciones base y utilidades.
- `lamudi_scraper_seleccionX.py`: Scripts principales para diferentes estados/tipos.
- `requirements.txt`: Dependencias del proyecto.

## Instalación
```bash
pip install -r requirements.txt
```
